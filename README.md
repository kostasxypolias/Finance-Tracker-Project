# Personal Finance Tracker

A simple Python-based finance tracker that allows users to record income and expenses through a terminal interface.

## Features
- Add income and expense transactions
- View transaction history
- Display account summary and balance
- Group expenses by category
- Remove the last transaction
- Includes automated test cases using pytest

## Technologies Used
- Python
- Object-Oriented Programming
- Command Design Pattern
- Pytest
